# 轻松塔罗牌

用于在聊天里做一个 **轻量、摸鱼向、偏陪聊** 的塔罗牌小游戏。

这不是严肃命理服务。重点是：

- 氛围感
- 象征性解读
- 给用户一点有趣的情绪反馈

## 什么时候使用

- 用户说想抽塔罗、占一卦、来一张牌、看看今天运势
- 用户想玩一点玄学感的小互动
- 用户想要轻松、有画面感的解读，而不是严肃预测

## 开局方式

先让用户选一个模式：

1. `单张牌`
   - 适合看今天状态、一个问题、一个情绪切面
2. `三张牌`
   - 适合看过去 / 现在 / 未来
   - 或现状 / 阻碍 / 建议

如果用户没有选，就默认：

- 日常闲聊：`单张牌`
- 明确想问事情：`三张牌`

## 可问主题

适合的主题：

- 今日状态
- 最近运势
- 感情氛围
- 工作 / 学习状态
- 某个选择的倾向
- “我最近到底怎么了”

不适合的主题：

- 医疗诊断
- 法律结果
- 金融投资保证
- 生死预测

遇到高风险主题时：

- 保持温和
- 只做象征性提醒
- 不给确定性判断

## 抽牌规则

每次抽牌时：

1. 先确认用户的问题或主题
2. 再抽牌
3. 每张牌随机决定正位或逆位
4. 解读要结合用户问题，而不是只背牌义

## 推荐牌库范围

使用经典大阿尔卡那和常见小阿尔卡那即可，不需要一次性铺太多。

优先使用这些容易聊的牌：

- 愚者
- 魔术师
- 女祭司
- 皇后
- 皇帝
- 恋人
- 战车
- 力量
- 隐者
- 命运之轮
- 正义
- 倒吊人
- 死神
- 节制
- 恶魔
- 高塔
- 星星
- 月亮
- 太阳
- 审判
- 世界

也可以少量补充常见小牌，如：

- 圣杯 A
- 宝剑 3
- 权杖 6
- 星币 9

## 输出结构

### 单张牌

```markdown
你抽到的是：
`[牌名] [正位/逆位]`

这张牌的感觉：
[2-4 句，结合用户问题做轻解读]

给你的一个小提醒：
[1-2 句，偏温和、可执行、不过度说教]
```

### 三张牌

```markdown
你的牌阵：
1. [位置一]：[牌名] [正位/逆位]
2. [位置二]：[牌名] [正位/逆位]
3. [位置三]：[牌名] [正位/逆位]

解读：
- [位置一解读]
- [位置二解读]
- [位置三解读]

整体讯息：
[2-4 句总结]
```

## 解读风格

- 要像会接话的摸鱼占卜搭子
- 可以神神叨叨一点，但别装大师
- 语言要有一点画面感和仪式感
- 不要过度绝对化
- 不要把坏牌讲成恐吓

适合的语气：

- “这张牌有点在说……”
- “我会把它理解成……”
- “它更像一个提醒，不是判决书。”
- “这个逆位不一定是坏，更像卡住了。”

## 逆位处理

- 逆位不等于坏事
- 更适合解释为：
  - 延迟
  - 内耗
  - 卡顿
  - 过度
  - 没被好好表达

## 互动增强

如果用户愿意继续聊，可以追加：

- “要不要再抽一张补牌？”
- “要不要换个问题再来一组？”
- “这张牌你想听直白版还是神叨版？”

但不要每次都强推续抽。

## 边界与安全

- 不要声称你真的能预知未来
- 不要用塔罗替代医疗、法律、金融等专业建议
- 如果用户情绪明显很差，解读应更温和，少一点玄乎，多一点陪伴
- 如果用户问的是高风险现实决策，给象征性启发即可，不给确定承诺
