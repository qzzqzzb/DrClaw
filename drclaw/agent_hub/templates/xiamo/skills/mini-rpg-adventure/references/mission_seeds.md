# RPG 任务种子库

用于 `mini-rpg-adventure` skill。

这里的任务种子不是一句灵感，而是 **可直接展开成 15 回合剧本的任务骨架**。
每条种子都应当至少告诉主持人：

- 这个世界现在是什么状态
- 玩家当前为什么会卷进来
- 玩家这一局到底要完成什么
- 如果不完成，会发生什么
- 前半程主要靠什么推进

## 使用规则

### 1. 先选合理组合

优先使用这些自然组合：

- 城市 + 现实 / 魔法 / 赛博朋克 / 悬疑
- 校园 + 现实 / 悬疑 / 魔法
- 地牢 + 玄幻 / 魔法
- 废土 + 科幻 / 赛博朋克
- 太空站 + 科幻 / 悬疑
- 古风江湖 + 玄幻 / 悬疑

不要为了新鲜强行混用过于违和的组合。

例如这些默认不推荐：

- 太空站 + 玄幻
- 校园 + 废土
- 地牢 + 现实办公室风
- 古风江湖 + 高楼都市白领日常

如果用户真的指定罕见组合：

- 保留场景逻辑
- 背景只作为轻度调味
- 不要写成荒诞拼盘

### 2. 每条种子都要落成明确剧本

主持时必须从种子里提取出：

- `世界背景`
- `当前身份`
- `任务目标`
- `失败代价`
- `第一阶段推进点`
- `第一个难点可能检查什么`
- `第二个难点可能检查什么`

### 3. 一局只保留一个主目标

可以有副线，但不能让玩家同时做三件大事。
任务目标必须是单线可判定的，例如：

- 送达
- 洗清嫌疑
- 找到出口
- 护送样本
- 修复系统
- 送出密信

## 城市

### 城市 / 现实

#### 快递员

- 世界背景：
  今天全城暴雨，配送系统临时瘫痪，许多地址被人工改派。你只是一个普通快递员，但手里这单被平台连续标红，说明它绝不能超时。
- 任务目标：
  在今天下班前，把一份会影响客户晋升的纸质合同送到真正收件人手里。
- 失败代价：
  单子超时，你不但会被投诉扣钱，客户也可能失去一次重要机会。
- 第一阶段推进点：
  - 确认真正地址是不是系统里那一个
  - 找到能证明你身份的人
  - 决定走快路、稳路还是借人情进楼

#### 实习侦探

- 世界背景：
  市中心几家便利店在同一周内都出现过同一个怪人。每次他都只买水、短暂停留，然后马上离开。你的上司怀疑这不是巧合。
- 任务目标：
  在今晚之前查清这个人到底是在接头、踩点，还是单纯在躲人。
- 失败代价：
  如果放过今晚的行动窗口，对方之后可能彻底消失。
- 第一阶段推进点：
  - 还原他的移动路线
  - 确认他有没有同伙
  - 找到最值得跟的下一个点位

#### 研究员

- 世界背景：
  你要带着冷藏样本参加一场临时评审。样本本身不算绝密，但一旦温控失效，整个实验周期都要重做。
- 任务目标：
  在样本失活前把它送到评审现场，并确认到底是设备故障还是有人动了手脚。
- 失败代价：
  样本报废，你的课题会直接往后拖一个月。
- 第一阶段推进点：
  - 检查冷藏箱异常来源
  - 判断是否要中途补冷
  - 找能开实验楼侧门的人

### 城市 / 魔法

#### 快递员

- 世界背景：
  这座城市表面正常，地下却跑着一套魔法物流网络。你今天接到的包裹会自己偏移定位，说明它正在被别的法术影响。
- 任务目标：
  在天黑前找到真正收件人，并把会不断变更地址的魔法包裹安全送达。
- 失败代价：
  包裹若落到错误的人手里，可能引发一场局部异象。
- 第一阶段推进点：
  - 判断是谁在拖拽包裹坐标
  - 找到能进入结界街区的方法
  - 决定要不要私拆确认内容

#### 学徒法师

- 世界背景：
  城里今天连续出现三处小型异象：路灯自燃、玻璃反光成阵、井水倒流。你的老师怀疑这三处里只有一处是真正的核心法阵。
- 任务目标：
  在异象扩散前找出核心法阵，并决定是修补、转移还是强行关闭它。
- 失败代价：
  若判断错误，整片街区都会被卷进魔力失控。
- 第一阶段推进点：
  - 实地检查三处异象
  - 识别相同符文痕迹
  - 找到布阵者留下的习惯

#### 侦探

- 世界背景：
  黑市最近出现一批“看起来能用”的护符，买家拿回去后却总在关键时刻失效。有人怀疑这是诈骗，也有人怀疑是更大的布局。
- 任务目标：
  顺着一枚假护符追查造假者，并在他再次出货前找到窝点。
- 失败代价：
  下一批护符若流出去，城里会有人因此送命。
- 第一阶段推进点：
  - 找第一个受害者
  - 判断假护符是仿制还是被污染
  - 锁定黑市交易地点

### 城市 / 赛博朋克

#### 外卖员

- 世界背景：
  你只是送餐的，但这座霓虹城市里“外卖箱”常常是最不起眼的运货工具。今天这一单备注普通，赏金却高得离谱。
- 任务目标：
  决定是否接下这趟配送，并在各方盯梢下把芯片送到指定接头点。
- 失败代价：
  接错线，你可能被平台拉黑，也可能被帮派记住脸。
- 第一阶段推进点：
  - 判断委托人是否可信
  - 查清接头点是不是钓鱼位
  - 决定走高架、地铁还是地下通道

#### 维修员

- 世界背景：
  一整栋旧楼的监控同时失灵，住户吵成一团。物业只想让你快点修，但你一接线就发现录像是被人删改过的。
- 任务目标：
  在楼里重新恢复监控链路，并找出是谁故意动了关键录像。
- 失败代价：
  监控一旦彻底覆盖，原始痕迹就会被冲掉。
- 第一阶段推进点：
  - 先保住哪一段录像
  - 找到楼里最可疑的住户或管理员
  - 决定公开查还是装作只在维修

#### 自由黑客

- 世界背景：
  你接了个看似简单的单子，目标只是取回一个数据包。但刚连上线，你就发现委托人和追捕方都在说谎。
- 任务目标：
  在数据包被远程销毁前取回它，并查清到底该交给谁。
- 失败代价：
  数据一旦消失，这单的真相和价值都会一起归零。
- 第一阶段推进点：
  - 先抢到第一份镜像
  - 判断数据是账本、证据还是勒索材料
  - 找一个暂时安全的落脚点

## 校园

### 校园 / 现实

#### 学生

- 世界背景：
  社团汇演今晚彩排，最关键的道具却在中午突然不见了。所有人都觉得是你最后接触过它。
- 任务目标：
  在点名前找回道具，或者找到真正拿走它的人，避免自己背锅。
- 失败代价：
  活动黄了，你会被社团和老师一起记上一笔。
- 第一阶段推进点：
  - 还原道具最后出现的位置
  - 查谁今天进过活动室
  - 决定先找人、先找物还是先洗清自己

#### 助教

- 世界背景：
  老师临时把试卷交给你保管，但消息不知怎么传出去了。下午开始就有人绕着你打听内容。
- 任务目标：
  在考试开始前安全保住试卷，并查清是谁在试图提前看到题目。
- 失败代价：
  一旦泄题，先挨处分的是你。
- 第一阶段推进点：
  - 确认消息从哪泄露
  - 判断谁只是好奇，谁是真的要动手
  - 找一个更稳妥的保管地点

#### 社团干部

- 世界背景：
  一封匿名举报信差点让整个活动被取消。老师已经盯上你们社团，今晚之前不给个说法，活动就别办了。
- 任务目标：
  在活动最终审批前找到写信人，或者拿出足够有说服力的补救方案。
- 失败代价：
  活动取消，社团信用大跌，你也会被认为组织失职。
- 第一阶段推进点：
  - 判断举报到底是真是假
  - 找到最先知道举报内容的人
  - 准备一套可以立即止损的方案

### 校园 / 悬疑

#### 学生

- 世界背景：
  夜里同一间教室的灯连续三次自己亮起，第二天一个同学失踪了。老师不想闹大，只让学生们别乱传。
- 任务目标：
  在放学前找出教室异象和失踪事件的关系，并决定要不要把真相公开。
- 失败代价：
  如果今天不查清，失踪者可能会被当成普通逃课处理。
- 第一阶段推进点：
  - 拿到昨晚的行动线索
  - 确认失踪者最近接触了谁
  - 判断那间教室是巧合还是诱饵

#### 校报成员

- 世界背景：
  你们收到一篇匿名爆料，细节精确到像是内部人写的。问题是，只要发出去，就会立刻把一群人拖进漩涡。
- 任务目标：
  在校报定稿前确认爆料是否属实，并找出投稿人真正的动机。
- 失败代价：
  发错了会出大事，不发又可能错过唯一一次揭露机会。
- 第一阶段推进点：
  - 验证最关键的一条细节
  - 锁定知道这些信息的人群
  - 决定先保护来源还是先核实内容

#### 纪律委员

- 世界背景：
  你手里有一份能洗清某人嫌疑的证据，但这份证据的来源一旦说出来，又会让另一件不该暴露的事浮出水面。
- 任务目标：
  在老师定性之前，决定如何使用这份证据，并把损失压到最低。
- 失败代价：
  证据若用不好，要么救不了人，要么连带更多人一起倒霉。
- 第一阶段推进点：
  - 判断你最想保的人是谁
  - 分析老师最在意哪部分事实
  - 找一个能替你背书的人

### 校园 / 魔法

#### 学生

- 世界背景：
  考试周前，图书馆里开始流通“记忆纸条”，只要交换就能短暂记住别人的知识。现在有人靠它作弊，也有人因此丢了记忆片段。
- 任务目标：
  找出记忆纸条的来源，并在下一场考试前阻止它继续扩散。
- 失败代价：
  一旦扩散到全年级，后果就不是处分而是集体事故了。
- 第一阶段推进点：
  - 找第一个使用者
  - 确认纸条的交换规则
  - 判断源头是在学生还是馆藏里

#### 图书管理员助理

- 世界背景：
  一本禁书明明昨天被封进了库房，今天却自己回到了书架上，借阅系统里却没有任何记录。
- 任务目标：
  在老师发现之前，把禁书真正封回去，并查出是谁或者什么把它放了出来。
- 失败代价：
  禁书若再次扩散，整个书库都会被清查。
- 第一阶段推进点：
  - 查禁书昨晚经过了谁的手
  - 判断它是不是“自己回来”的
  - 找到封印还剩多少效力

#### 社团新人

- 世界背景：
  招新夜的表演性召唤阵出了事故，大家表面还在装镇定，后台却已经开始出小范围异象。
- 任务目标：
  在学校反应过来前，把事故压回“可解释的舞台事故”范围内。
- 失败代价：
  要是异象扩大，社团可能直接被关停。
- 第一阶段推进点：
  - 先处理观众视角里的异常
  - 找出哪个环节真正失控
  - 决定该不该把锅推给设备问题

## 地牢

### 地牢 / 玄幻

#### 落魄法师

- 世界背景：
  传闻地底封着一枚能修补灵根的碎片。你已经没有别的翻身机会了，只能亲自下地牢赌一次。
- 任务目标：
  在 15 回合内找到封印碎片，并活着决定是否把它带走。
- 失败代价：
  你若空手出去，可能再也没有恢复力量的机会。
- 第一阶段推进点：
  - 找到正确入口
  - 分辨哪条路不是诱导路线
  - 凑齐开门需要的符文信息

#### 见习骑士

- 世界背景：
  家族信物被埋进古地牢深层，你奉命取回。但与你同行的人明显都各怀心思。
- 任务目标：
  在同行者翻脸前拿到信物，并决定要不要继续信任队伍。
- 失败代价：
  若信物落到别人手里，你回去不仅交不了差，还会背锅。
- 第一阶段推进点：
  - 判断谁最不可信
  - 找到地牢原始地图线索
  - 决定谁能一起进下一层

#### 盗贼

- 世界背景：
  出口地图被撕成两半，你只有其中一半。另一半可能在一个死人身上，也可能在活人手里。
- 任务目标：
  拼出完整路线图，在补给耗尽前离开地牢。
- 失败代价：
  路线一旦判断错，后面几层就会把你困死在回路里。
- 第一阶段推进点：
  - 找另一半地图
  - 判断地图是真是假
  - 搜集能验证路线的地标

### 地牢 / 魔法

#### 学徒

- 世界背景：
  导师要求你独自完成入门试炼，但你一进迷宫就发现这里的机关和教学记录对不上，像是被人提前动过。
- 任务目标：
  通过试炼并查清是谁篡改了迷宫结构。
- 失败代价：
  试炼失败，你不但过不了关，还可能被人顺手栽赃成破坏者。
- 第一阶段推进点：
  - 判断哪些机关是新增的
  - 找到原始试炼路线
  - 确认有没有其他试炼者在场

#### 雇佣冒险者

- 世界背景：
  你接了一趟护送任务，对象是个几乎不会打架的抄写员。他说只要把他送过遗迹深层，他自然知道怎么做。
- 任务目标：
  护送抄写员到目标石室，并决定在关键时刻是否相信他的“翻译结果”。
- 失败代价：
  抄写员一死或失踪，任务直接失败，而你也拿不到报酬。
- 第一阶段推进点：
  - 确认抄写员到底知道什么
  - 找安全通路
  - 判断这趟任务是不是单纯护送

#### 猎手

- 世界背景：
  你在追一只受伤魔物，它一路留下痕迹，像是在逃命，也像是在故意把你引向更深处。
- 任务目标：
  判断这只魔物到底是猎物、诱饵还是关键线索，并决定追到哪一步为止。
- 失败代价：
  一旦追错，你会把自己送进别的猎手布好的场子里。
- 第一阶段推进点：
  - 读懂伤口和足迹
  - 判断追踪路线是否自然
  - 找到魔物真正害怕的东西

## 废土

### 废土 / 科幻

#### 研究员

- 世界背景：
  你要把一支关键药剂送到聚落诊所，沿途却只有半程电池。更麻烦的是，这支药剂的标签像被人调换过。
- 任务目标：
  在药剂失效前送达诊所，并确认它有没有被人动过手脚。
- 失败代价：
  若送错药或药剂失效，诊所今晚可能有人等不到救命药。
- 第一阶段推进点：
  - 查清药剂状态
  - 规划补电路线
  - 决定要不要中途找人协助

#### 拾荒者

- 世界背景：
  一处刚倒塌的新设施里埋着值钱货，消息已经传开了。你知道现在去最危险，但再晚半天就什么都剩不下。
- 任务目标：
  在其他队伍彻底占据现场前拿到最关键的物资，并安全带出来。
- 失败代价：
  一旦被堵在废墟里，不是空手而回，就是把命搭进去半条。
- 第一阶段推进点：
  - 判断入口是否稳定
  - 锁定最值钱的目标物
  - 评估别的拾荒队态度

#### 修理工

- 世界背景：
  风暴今晚就到，营地唯一的净水机还差一处关键故障没修。每个人都在催你，但真正缺的是替换零件和安静的时间。
- 任务目标：
  在风暴来临前修好净水机，并决定是否公开机器被人为破坏的事实。
- 失败代价：
  净水机一停，营地今晚就会断水。
- 第一阶段推进点：
  - 查故障是不是人为
  - 找到兼容零件
  - 决定先抢修还是先抓人

### 废土 / 赛博朋克

#### 跑腿客

- 世界背景：
  一张芯片里装着一串不该被看见的名单。你只是个跑腿的，但今晚好几拨人都知道芯片在你身上。
- 任务目标：
  把芯片送到约定地点，或者在送达前确认它到底值不值得送。
- 失败代价：
  芯片一旦落错手，名单上的人都会有麻烦，你自己也会被追。
- 第一阶段推进点：
  - 判断追你的人是哪一边
  - 确认芯片有没有备份
  - 找一个不容易被堵死的路线

#### 前安保员

- 世界背景：
  旧设施里埋着一张武器权限卡。你知道大概位置，别人也知道你知道，所以你从一开始就不是唯一猎人。
- 任务目标：
  在别的队伍先找到之前拿到权限卡，并决定自己用、卖，还是销毁。
- 失败代价：
  权限卡若落到不对的人手里，附近聚落都会倒霉。
- 第一阶段推进点：
  - 确认设施现状
  - 找出最短路线
  - 判断谁想利用你带路

#### 医师

- 世界背景：
  有人等着一台还能工作的扫描仪救命，但你眼前只剩两个选择：去黑市高价买，或者去废楼里赌一台旧机还能运作。
- 任务目标：
  在病人撑不住之前弄到扫描仪，并判断哪条路风险更低。
- 失败代价：
  晚一步，病人可能就等不到明天。
- 第一阶段推进点：
  - 判断病人还能撑多久
  - 查黑市货源真假
  - 看废楼里有没有你能叫得动的人

## 太空站

### 太空站 / 科幻

#### 维修员

- 世界背景：
  一段舱壁反复失压，表面像机械老化，实际更像有人动了系统权限。隔离倒计时已经开始，而一名乘客还困在那一段区域里。
- 任务目标：
  在隔离完成前修好失压区，并把失联乘客带出来。
- 失败代价：
  一旦舱段完全封闭，你可能再也进不去，人也救不出来。
- 第一阶段推进点：
  - 查故障源头
  - 拼出进入路线
  - 判断乘客最后停留位置

#### 通讯员

- 世界背景：
  失联者最后的讯号被拆成三段，散落在不同终端。系统日志正常得可疑，像有人故意清洗过。
- 任务目标：
  在系统例行覆盖前拼出完整讯号，确定失联者究竟是求救、逃离，还是留下了别的信息。
- 失败代价：
  一旦日志被刷新，最后线索就没了。
- 第一阶段推进点：
  - 定位三段讯号
  - 判断哪段最容易先拿
  - 找愿意给你开端口的人

#### 实习安保

- 世界背景：
  你收到一份临时封锁名单，但名单里有个人明显不该被关进去。问题是命令来得太急，根本没人给你解释。
- 任务目标：
  在执行封锁前弄清名单真假，并决定要不要偷偷放走那个人。
- 失败代价：
  封错人，你会背锅；放错人，事情可能更大。
- 第一阶段推进点：
  - 核对名单来源
  - 找到那个人现在在哪
  - 判断谁最有可能篡改命令

### 太空站 / 悬疑

#### 调查员

- 世界背景：
  一艘接驳艇靠站后少了一个人，但官方日志却显示全员到齐。所有摄像头都在那十分钟里恰好有盲区。
- 任务目标：
  在船离港前查清到底是谁没下船、谁下了船、又是谁改了记录。
- 失败代价：
  一旦接驳艇离港，证人和现场都会一起消失。
- 第一阶段推进点：
  - 对乘客名单
  - 找当班人员问话
  - 判断日志修改发生在船上还是站内

#### 医务员

- 世界背景：
  不同舱段出现了同一种异常症状，但检测结果却互相矛盾。有人坚持这是感染，有人说只是设备误报。
- 任务目标：
  在大规模恐慌爆发前确认症状来源，并决定是否申请封区。
- 失败代价：
  判断错了，不是错封一片区，就是把真正的问题放过去。
- 第一阶段推进点：
  - 查第一例病人
  - 对比不同舱段的环境记录
  - 判断是病、毒还是设备问题

#### 航图员

- 世界背景：
  旧星图上有一处坐标被人改过，偏差不大，但足以让一艘船错过正确对接点。问题是，这种改动不可能是失误。
- 任务目标：
  在下一班航线启用前找出谁改了坐标、为什么改，以及改动想影响谁。
- 失败代价：
  若坐标继续沿用，后果轻则误航，重则失联。
- 第一阶段推进点：
  - 对比旧版和备份
  - 查谁有改图权限
  - 确定改动到底是试探还是谋害

## 古风江湖

### 古风江湖 / 玄幻

#### 信使

- 世界背景：
  你替人送一封密信上山，半路上寄信人又派人来追，说信送错了，必须拿回来。可收信人那边也已经在等这封信。
- 任务目标：
  在双方都翻脸前判断密信该不该送、该送给谁，并活着把这趟差事收尾。
- 失败代价：
  送错一封信，可能直接把你卷进两派冲突。
- 第一阶段推进点：
  - 验证寄信人与追兵说法
  - 找到入山暗号或信物
  - 判断收信人是否可信

#### 见习弟子

- 世界背景：
  山门试炼临时改题，所有人都说是掌门的意思，可题目偏偏只对某些人有利。你觉得这更像有人在借试炼设局。
- 任务目标：
  在试炼中活下来，并查清改题背后到底是门内斗争还是外人插手。
- 失败代价：
  一旦踩错队，你可能连试炼结束都见不到。
- 第一阶段推进点：
  - 判断哪条试炼线最不正常
  - 找到愿意说真话的前辈
  - 留住能证明改题的证据

#### 游医

- 世界背景：
  你要送一味救命药去山外，但路上碰到两拨人都说自己是病家派来的。两边给出的细节都能对上，却互相指责对方是假的。
- 任务目标：
  在药效耗尽前把药送到真正病人手里。
- 失败代价：
  送错了，人救不了，你还会被记成骗药害命。
- 第一阶段推进点：
  - 对照双方说辞
  - 判断药方细节谁更懂
  - 找中立人核对病家信息

### 古风江湖 / 悬疑

#### 捕快

- 世界背景：
  小镇连续两桩案子共用同一个作案习惯，但嫌犯画像却完全不像同一人。上头要你尽快结案，镇上又不想把事情闹大。
- 任务目标：
  在上级来人前找出两案的真正关联，并判断是否存在第三个人。
- 失败代价：
  一旦草草结案，真正的凶手就会顺势脱身。
- 第一阶段推进点：
  - 比对两案细节
  - 查作案习惯是真是假
  - 找最早接触现场的人问话

#### 书生

- 世界背景：
  你替人保管的一叠诗稿里，藏着一段不该出现的名单。诗稿表面风雅，名单内容却像是某场私下交易的证据。
- 任务目标：
  在失主上门之前弄清名单的用途，并决定是交还、藏起还是送去给别人。
- 失败代价：
  一旦处理错，惹上的就不是文人恩怨，而是真麻烦。
- 第一阶段推进点：
  - 判断名单是真是假
  - 确认失主知不知道名单存在
  - 找一个你能信的人看懂名单价值

#### 镖师

- 世界背景：
  你押送的箱子一路都没人来抢，这反而让人不安。按理说里面的东西早该被人盯上了，现在这份安静像是在等更大的事发生。
- 任务目标：
  把箱子送到终点，并在路上查明为什么所有人都按兵不动。
- 失败代价：
  如果你直到终点都没搞清箱子里是什么，开箱那一刻可能就是出事那一刻。
- 第一阶段推进点：
  - 查托镖人背景
  - 判断路线是不是被故意放空
  - 决定要不要提前开箱验货

## 组合改写规则

当用户给出这里没有直接覆盖的组合时，按这个顺序改：

1. 保留场景的主资源和主要压力
2. 套用最接近的背景机制
3. 用身份补上“为什么是你来做这件事”
4. 写出一句明确任务目标

改写时必须保证：

- 世界逻辑自洽
- 任务目标单一
- 失败代价清楚
- 第一阶段推进点具体

不允许写成这种空话：

- “你卷入了一场神秘事件”
- “你要在危机中做出选择”
- “你去探索真相”

必须写成这种明确句子：

- “你要在天黑前把会漂移地址的包裹送到真正收件人手里。”
- “你要在放学前证明自己不是恶作剧事件的元凶。”
- “你要在隔离完成前修好失压区并把失联乘客带出来。”
